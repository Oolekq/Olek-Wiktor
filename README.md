# Olek-Wiktor

Projekt służy do walidacji i przetwarzania danych wejściowych przy użyciu Pydantic.
Dane wejściowe są sprawdzane pod kątem poprawności, typów oraz reguł biznesowych.
System potrafi automatycznie konwertować dane (np. string na datę lub liczbę).

Instalacja:
```
pip install-r requirements.txt
```
Uruchomienie:
```
python main.py
```
